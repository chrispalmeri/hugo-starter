# hugo-starter

Basically just a Hugo site, so see [Hugo][1] with questions. Adding my own commonly used defaults and customization and will document later.

  [1]: https://gohugo.io
  [2]: https://nodejs.org

## setup

Uses "hugo-bin" npm module rather than the Windows executable, so:

`npm install`

## commands

`npm run serve`

`npm run build`

`npm run clean`
